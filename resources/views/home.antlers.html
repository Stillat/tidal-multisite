Tidal, a powerful documentation starter kit for Statamic.

{{ svg:hero-icons/outline/wrench-screwdriver class="w-4 h-4 mr-4 inline" /}} Multiple projects

Have many different projects you want to document and manage from a single site? No problem. Tidal supports multiple projects.

{{ svg:hero-icons/outline/lifebuoy class="w-4 h-4 mr-4 inline" /}} Multiple versions

Tidal provides a framework for managing documentation across multiple versions of a single project, helping you to keep things organized.

{{ svg:hero-icons/outline/globe-alt class="w-4 h-4 mr-4 inline" /}} Multiple languages

Tidal implements its multi project/version features using regular Statamic collections so you can still use Statamic's multi-site features. Enjoy!
